# Meta Team Match Docs Index

Use this file to quickly find the right note before recruiter or hiring-manager conversations.

## Recommended order

1. `docs/META_TEAM_MATCH_HM_ONE_PAGER.md`  
   Open this right before the HM call.

2. `docs/META_TEAM_MATCH_HM_SCRIPT.md`  
   Use this during the call if you want a structured question flow and scoring sheet.

3. `docs/META_TEAM_MATCH_OWNERSHIP_CHECK.md`  
   Use this if you want a focused filter for one question: real platform ownership or support-heavy slice?

4. `docs/META_TEAM_MATCH_GUIDE.md`  
   Read this for the full picture: team scope, metrics, influence, growth, LLM usage, and HM matrix.

5. `docs/META_TEAM_MATCH_7_DAY_CRASH_COURSE.md`  
   Use this if you want a short learning plan before the conversation.

6. `docs/META_TEAM_MATCH_NOTES.md`  
   Use this for more personal narrative prep, spoken talk tracks, and positioning adjacent platform experience.

- `docs/META_TEAM_MATCH_7_DAY_CRASH_COURSE.md`

## Suggested usage

- **Before the HM call:** read the One-Pager, then skim the Script
- **If you mainly want to test role shape:** read the Ownership Check
- **If you have 30–60 minutes:** read the Guide
- **If you have a few days:** follow the 7-Day Crash Course
- **If you want to improve your story:** read the Notes file

## Short summary of the whole set

The docs together cover:

- what the team likely does
- why it can be good for E5 -> E6 growth
- major infra problems and important metrics
- how LLMs may fit into the team
- how to assess influence, ownership, and visibility
- how to distinguish a strong platform seat from a support-heavy slice
- what questions to ask the hiring manager
- how to ramp quickly if you are new to ML infra
